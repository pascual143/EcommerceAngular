# MongoDB local:           
1) Iniciar el servicio MongoDB (si no está en ejecución):

2) verificar el estado para asegurarte de que esté en ejecución: 
         sudo service mongodb status
3) Detener el servicio MongoDB: 
         sudo service mongodb stop
